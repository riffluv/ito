## エグゼクティブサマリー

「序の紋章III（オンライン版）」は、**技術的完成度が極めて高い個人開発のオンライン協力ゲーム**である。Firebase RTDB Presenceシステムの実装は業界トップクラスであり、Next.js 14/Pixi.js 8/XState 5といった最新技術スタックを高度に活用している。

ただし、**市場性とマネタイズ実績がゼロ**であることから、売却適正価格は技術的価値の30-50%程度に留まる。企業がゼロから同等品を開発する「再現コスト」と、実際に市場で取引される「適正価格」には大きな乖離がある点に留意が必要。

**結論**: 技術IPとしての価値は高いが、事業資産としての価値は限定的。ライセンス販売またはホワイトレーベル化が最も現実的な収益化パス。

## 2. 技術的独自性の評価

### 2.1 RTDB Presenceシステム ⭐⭐⭐⭐⭐ (5/5)

**評価**: **業界トップクラスの実装。Firebase公式サンプルを大きく超える品質。**

#### 実装の特徴：
- **複数タブ対応**: connId単位でPresence管理（Firebase公式では単純なuid管理のみ）

- **sendBeacon API**: ページ離脱時の確実なハートビート送信
- **Graceful degradation**: presenceDegraded状態での段階的機能低下
- **Stale connection cleanup**: 残存接続の自動削除
- **メトリクス収集**: 詳細なテレメトリー

#### 市場比較：
- Firebase公式ドキュメントのサンプルコード: 50-100行程度

- Among Us、Fall Guysなどのプレゼンスシステムと同等以上

#### 価値評価：
- **単体IPとしての価値**: **150-300万円**
- エンタープライズ向けSDKとして販売可能なレベル

## 8. 査定根拠と計算式

### 8.1 再現コスト計算

```
再現コスト = 人件費 + インフラコスト + ツールコスト

標準構成の場合:
人件費 = (フロントエンドSr 500万 + バックエンドMd 240万 + デザイナー 210万 +
         ディレクター 360万 + QA 120万 + PM 80万) = 1,510万円
インフラ + ツール = 50万円
合計 = 1,560万円
```

### 8.2 売却適正価格計算

```
売却価格 = 再現コスト × 減価率 + 技術IP価値 - リスク減算

標準評価の場合:
再現コスト × 40% = 1,560万 × 0.4 = 624万円
技術IP価値 = 230万円
ドキュメント等 = 100万円
リスク減算（マネタイズなし） = -200万円
合計 = 754万円 ≒ 800万円
```

### 8.3 ライセンス価格計算

```
年間ライセンス = (売却価格 / 想定利用年数) × 利益率

非独占の場合:
800万円 / 10年 × 1.5倍（複数社販売） = 120万円/年
```
