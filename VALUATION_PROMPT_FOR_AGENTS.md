# AI Agent向け プロジェクト査定プロンプト

このプロンプトを他のAIエージェント（Claude、ChatGPT、Gemini等）に投げることで、客観的なプロジェクト査定を得られます。

---

## 📋 基本プロンプト（短縮版）

```
このリポジトリのコードを読んで、技術的な市場価値を査定してください。

【条件】
- 既存の査定ドキュメント（VALUATION_REPORT.md等）は無視
- 実際のコードを読んで判断
- お世辞なしで、技術的事実のみで評価
- 日本の個人開発市場を基準に査定

【確認してほしいポイント】
1. lib/state/roomMachine.ts - XState FSM実装
2. lib/firebase/presence.ts - RTDB Authoritative Presence
3. lib/telemetry/safeUpdate.ts - PWA Safe Update
4. components/ui/GameCard.tsx - Pixi.js + GSAP統合
5. app/api/rooms/[roomId]/reset/route.ts - Server-Authoritative設計
6. firestore.rules - セキュリティルール
7. tests/ - テストカバレッジ
8. docs/ - ドキュメント品質

【開発期間】
- 3ヶ月（90日）
- 1,174コミット（平均13コミット/日）

【技術スタック】
Next.js 14 / TypeScript 5.9 / Chakra UI v3 / Pixi.js 8 / GSAP 3 / XState 5 / Firebase / Stripe

この情報をもとに、以下を回答してください：
1. 技術的完成度（10点満点）
2. 推定市場価値（円建て）
3. 同レベルを企業が開発した場合のコスト
4. 日本の個人開発ブラウザゲームでの順位（TOP何%か）
5. 3ヶ月でこの品質を達成することの異常性
```

## 注意事項

1. **エージェントごとに査定が変わる可能性**
   - Claude: 保守的（技術的事実重視）
   - ChatGPT: 楽観的（市場性重視）
   - Gemini: 中立的

2. **コンテキスト制限**
   - 大規模リポジトリは全ファイル読めない
   - 重要ファイルを明示すること

3. **比較時の注意**
   - 複数エージェントの中央値を採用
   - 極端な外れ値は無視
