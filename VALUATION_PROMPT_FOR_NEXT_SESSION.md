## 査定手法

### 1. コストアプローチ（技術再現コスト）

```
人件費 + 間接費 + 試行錯誤コスト = 技術資産価値
```

### 2. マーケットアプローチ（類似取引比較）

```
類似ゲームのM&A事例
インディーゲーム評価倍率（売上の2-4倍）
```

### 3. インカムアプローチ（DCF法）

```
将来キャッシュフロー割引現在価値
割引率: 15%（スタートアップ標準）
成長率: 3%（永続）
```
